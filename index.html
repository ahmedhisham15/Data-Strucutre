Stack Mastery

  • Stack = LIFO (Last In, First Out). All operations happen at TOP.
  • Core ops: push, pop, top, empty, size (and full if array-based fixed size).
  • Tracing: draw the stack after every operation.
  • Applications: reverse string, balanced parentheses, expression conversion & evaluation.

Expressions

  • Precedence: ^ then * / then + -
  • Infix: A + B (human-friendly)
  • Postfix: A B + (easy for computer)
  • Prefix: + A B

Queue + Circular Queue

  • Queue = FIFO (First In, First Out). Enqueue at REAR, Dequeue from FRONT.
  • Circular queue uses wrap-around: (i+1) % size
  • Exam loves front/rear tracing.
